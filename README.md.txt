# 📊 Ecommerce India Sales Dashboard

### 🎯 Objective
To analyze and visualize e-commerce sales performance using Power BI.

### 📈 Key Insights
- Top Categories: Sports & Home
- Highest Revenue Month: May
- Region-wise insights available through slicers
- YoY trends for both Sales and Profit

### ⚙️ Tools & Technologies
- Power BI Desktop
- Microsoft Excel
- DAX (Data Analysis Expressions)
- Power Query

### 🧩 Features
- Interactive filters by region
- KPI Cards: Total Sales, Profit, Orders, Avg Order Value
- Category-wise and Sub-category insights
- YoY sales and profit comparison

### 🚀 Result
A fully interactive business dashboard to support data-driven decisions.
